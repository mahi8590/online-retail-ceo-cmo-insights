# Online Retail CEO & CMO Insights

This project presents a Power BI dashboard built on online retail data
to provide actionable insights for executive decision-makers (CEO & CMO).

## Key Objectives
- Analyze monthly revenue trends (2011)
- Identify top revenue-generating countries (excluding UK)
- Identify top customers by revenue
- Visualize demand distribution by country

## Tools Used
- Power BI
- Power Query (Data Cleaning)
- DAX (Revenue Calculation)

## Files Included
- `Online_Retail_CEO_CMO_Insights.pbix` – Power BI dashboard
- `screenshots/` – Dashboard visuals
- `dataset_info/` – Dataset description
- `insights/` – Key findings summary
